TROUBLESHOOTING:
================================================================================

Problem: "Sitemap not found" error in Search Console
─────────────────────────────────────────────────
Solution:
1. Check robots.txt has correct sitemap URL
2. Verify sitemap.xml XML format is valid
3. Check domain in sitemap matches submitted domain
4. Resubmit sitemap

Problem: URLs not indexed
──────────────────────────
Solution:
1. Check Google Search Console for crawl errors
2. Verify noindex meta tags absent
3. Check robots.txt allow rules
4. Submit URL directly in Search Console

Problem: Domain changed
──────────────────────
Solution:
1. Update all URLs in sitemap.xml
2. Add 301 redirects from old domain (if applicable)
3. Update robots.txt
4. Resubmit to Search Console
